## 功能模块

| 模块 | 内容 |
|------|------|
| ☀️ 作息打卡 | 全天13个时间节点，一键打勾，热力图追踪 |
| 💪 训练打卡 | 分3阶段自动切换，每个动作有 SVG 图文说明 |
| 💰 记账 | 4维分类（必要/不必要/情绪/人情）+ 营养进度条 |
| 📊 统计 | 训练热力图、月支出分析、里程碑进度 |
| 📋 导出 | 一键复制 Markdown 到飞书，或下载 CSV |

## 技术特点

- **纯静态 HTML**，无需服务器，无需登录
- **数据存本地浏览器**（localStorage），完全私密
- **手机优先设计**，添加到桌面像原生App
- 可开源，可迁移到微信小程序

## 部署到 GitHub Pages（免费）

### 第一次部署

1. 登录 [github.com](https://github.com)，点右上角 **+** → **New repository**
2. 仓库名填：`muyu-life`（或你喜欢的名字）
3. 设置为 **Public**（GitHub Pages 免费版需要公开）
4. 点 **Create repository**

5. 在仓库页面点 **uploading an existing file**
6. 把 `index.html` 拖进去，点 **Commit changes**

7. 进入仓库 **Settings** → 左侧 **Pages**
8. Source 选 **Deploy from a branch** → Branch 选 **main** → 文件夹选 **/ (root)**
9. 点 **Save**

10. 等1-2分钟，刷新页面，会出现访问地址：
    ```
    https://你的用户名.github.io/muyu-life/
    ```

### 添加到手机桌面

**安卓（Chrome）：**
1. 手机打开上面的网址
2. 浏览器右上角菜单 → **添加到主屏幕**
3. 桌面出现图标，点击直接打开，体验≈小程序

## 数据说明

所有打卡和记账数据保存在**本地浏览器**，不会上传到任何服务器。
换设备或清除浏览器缓存会导致数据消失，建议定期「导出 Markdown」备份到飞书。
